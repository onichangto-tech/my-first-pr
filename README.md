# My First PR

このプロジェクトは GitHub のプルリクエストを練習するためのリポジトリです。

## はじめかた

```bash
# 1. リポジトリをクローンする
git clone https://github.com/onichangto-tech/my-first-pr.git
cd my-first-pr

# 2. ブランチを作る
git checkout -b my-feature

# 3. 変更をコミットする
git add .
git commit -m "Add my feature"

# 4. プルリクエストを出す
git push origin my-feature
```

## 貢献について

バグ報告や改善提案は [Issue](https://github.com/onichangto-tech/my-first-pr/issues) からお知らせください。
